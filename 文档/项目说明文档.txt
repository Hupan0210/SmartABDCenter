南来北往·智能ABD中控系统 项目说明文档
==========================================

【项目简介】
该项目整合了网络状态检测、局域网设备扫描、ADB设备管理、模拟遥控器操作、文件上传APK安装与卸载、远程剪贴板以及设备监控画面等模块。整体系统旨在为使用者提供一个智能化的中控平台，操作简单、实时刷新、便于扩展。

【目录结构说明】
1. 前端：包含 index.html、js/main.js、css/style.css 等，用于展示各项功能模块并与后端交互。
2. 后端：采用 Node.js + Express 搭建，提供各模块相应API（如网络IP获取、局域网扫描、ADB操作等）。
3. 脚本：提供一键启动脚本（启动.cmd），在Windows环境下实现一键检查、依次启动相关程序。
4. 文档：项目整体说明与使用说明文档。
5. package.json：npm依赖配置文件及启动命令。

【主要功能说明】
1. 网络状态：自动获取并显示本机局域网IP地址。
2. 局域网设备扫描：使用 arp 命令扫描局域网内的设备，展示设备IP及预留的名称、型号信息。
3. ADB设备：调用ADB命令，获取已开启ADB的设备列表，并提供一键连接功能（adb connect）。
4. 模拟遥控器：前端JS发送控制指令（确认、返回、音量控制、频道控制、数字按键），后端通过adb shell input keyevent实现控制。
5. 文件上传/APK安装与卸载：通过文件上传控件上传APK文件，后端使用adb命令安装或卸载应用。
6. 远程剪贴板：将用户输入内容通过后端转发至设备（示例中采用文件推送方法）。
7. 已连接设备监控画面：调用adb截屏命令获取设备当前屏幕，并定时刷新显示。

【依赖安装】
在项目根目录下执行：
    npm install

【启动项目】
建议使用 脚本/启动.cmd 文件一键启动，或者依次执行：
    npm install
    npm start
随后在浏览器中访问：http://localhost:3000
